### 1. DOM은 어떤 말의 약자인가요?

ans. `Document Object Model`

### 2. HTML 문서가 완전히 로드된 후에(DOM이 완전히 구성된 후에) 자바스크립트 코드를 실행할 때 가장 적절한 이벤트는 무엇인가요?

ans : `DOMcontentLoaded`

### 3. 자바스크립트로 스타일을 직접 변경하려면 어떤 속성을 사용해야 하나요?

ans : `요소.style`

### 4. `<ul>` 요소 내부에서 새로운 `<li>` 요소를 추가할 때 자바스크립트에서 사용하는 메소드는 무엇인가요?

ans : `appendChild()`

### 5. DOM에서 요소의 텍스트 내용을 변경할 때 사용할 수 <u>없는 속성</u>은 무엇인가요?

ans : `textHTML` (`textContent`, `innerText`, `innerHTML` 가능)

### 6. 체크 표시를 누르면 항목 텍스트가 회색으로 변하면서 글자 가운데에 가로줄이 그려지는 코드 작성

ans : `quiz-1.html` 에 작성

### 7. 행과 열의 개수를 입력받아 표 만들기 코드 작성

ans : `quiz-2.html` 에 작성
